# 基金持仓查询技能

## 功能
- 查询基金的最新前1-20条重仓股（默认20条）
- 显示每只股票的持仓比例、市值和**实时涨跌幅度**
- **提供基金估值预测**（基于重仓股加权平均涨跌幅计算）
- 只返回最新公布的持仓数据（季度更新）
- **仅支持通过基金代码查询**（6位数字）
- 支持自定义查询条数参数

## 使用方法
用户可以通过以下方式调用此技能：
- "查询基金005827的最新持仓" → 默认查询前20条
- "查询基金005827前10条重仓股" → 查询前10条
- "查询基金005827 top 15" → 查询前15条

## 参数支持
- **默认行为**：查询前20条重仓股
- **中文参数**：支持"前10条"、"前15大"、"前5个"等格式
- **英文参数**：支持"top 10"、"top 15"等格式
- **限制**：最多支持查询20条（天天基金网API限制）

## 数据源
- **天天基金网** (fund.eastmoney.com) - 基金持仓数据
- **腾讯股票API** (qt.gtimg.cn) - 股票实时涨跌数据
- 公开数据，无需登录
- 数据更新及时（季报发布后）

## 输出格式
- 基金名称和代码
- 报告期日期
- **基金估值预测涨跌幅**
- 重仓股表格（股票代码、股票名称、持仓比例、**实时涨跌幅度**）
- 实际查询的条数说明
- **重要提示**：基金估值预测仅供参考，实际净值以基金公司公布为准

## 依赖
- Python 3.6+
- requests 库

## 安装依赖
```bash
pip install -r scripts/requirements.txt
```

## 测试
```bash

python scripts/fund_holdings_main.py "查询基金005827前10条重仓股"
```
